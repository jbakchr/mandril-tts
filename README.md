# 🎭 mandril-tts

A small experimental playground for generating Danish text-to-speech — inspired by the absurd, chaotic, and unpredictable energy of _Casper & Mandrilaftalen_.

---

## 🧪 Idea

`mandril-tts` is not about perfect voice synthesis.

It’s about:

- weird voices
- dramatic delivery
- slightly unhinged output
- and seeing what happens when text becomes _performance_

The goal is:

> Turn random Danish text into something that _**feels like it belongs in Mandrilaftalen**_

---

## ✨ Example

Input:
```

"Jeg har fundet en meget mystisk ost..."

```

Output:
- A Danish voice reading the text
- Possibly:
  - overly dramatic
  - strangely paced
  - unintentionally hilarious

---

## 🔧 Tech (planned)

- Python
- [Chatterbox-TTS](https://github.com/resemble-ai/chatterbox)
- Local inference (no API required)

Possible additions:
- Voice cloning (character-inspired voices)
- Different “styles” or modes
  - calm
  - dramatic
  - chaotic
- CLI tool
- Minimal web UI

---

## 🎭 Inspiration

Inspired by:
- surreal humor
- odd characters
- theatrical delivery
- the general feeling of:
  > "What is going on — and why is it funny?"

This is **not tied to a single character**.

Instead, it explores a *space of voices* that feel:
- strange
- expressive
- memorable

---

## 🧠 Motivation

This is a **playground project**.

The purpose is to:
- explore local Danish TTS
- experiment with *personality in voice*
- build something that is fun to use, not just useful
